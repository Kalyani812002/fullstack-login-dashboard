# 🚀 Full Stack Login Dashboard

## 📌 Project Overview
This is a simple full stack web application that demonstrates user authentication and dashboard functionality.  
The application allows users to log in using email and password, and upon successful login, they are redirected to a dashboard displaying dummy data.

---

## 🚀 Features
- 🔐 User Login (Email & Password)
- ✅ Basic Form Validation
- 📊 Dashboard with Dummy Data (Leads, Tasks, Users)
- 🔄 Backend API Integration
- 🚪 Logout Functionality

---

## 🛠 Tech Stack

### Frontend:
- React.js

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB (Optional / Static for demo)
